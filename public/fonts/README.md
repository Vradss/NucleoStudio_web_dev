# Fuentes

Coloca aquí los archivos de fuentes en formato `.woff2`:

1. **MonaSansNarrow-Black.woff2** - Para títulos (Mona Sans Narrow Black)
2. **SpaceMono-Regular.woff2** - Para tags (Space Mono Regular)
3. **MonaSans-Regular.woff2** - Para subtítulos (Mona Sans Regular)

## Formatos soportados

Si tienes las fuentes en otros formatos (`.ttf`, `.otf`, `.woff`), puedes convertirlas a `.woff2` usando herramientas como:
- [CloudConvert](https://cloudconvert.com/)
- [Font Squirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)

## Estructura esperada

```
public/fonts/
├── MonaSansNarrow-Black.woff2
├── SpaceMono-Regular.woff2
└── MonaSans-Regular.woff2
```
